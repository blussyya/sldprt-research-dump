# Evidence Archive

Append-only archive for raw outputs, forensic dumps, script summaries, and copied evidence that supports project-wide conclusions.

See `../EVIDENCE_PRESERVATION_POLICY.md` for the repository-wide evidence rules.

Do not delete historical evidence. If evidence is superseded, add a new file and reference the old file from the new note.

Suggested filename pattern:

```text
YYYY-MM-DD_exp-NNN_short-description.md
YYYY-MM-DD_script-output_short-description.txt
```

Each evidence file should include:

```text
Source script or method:
Exact command:
Script path:
Input files:
Corpus size:
Faces/models tested:
Raw output:
Related experiment:
Related conclusion:
Date produced:
Date captured:
Known gaps:
```
